## Usage

```
iMac$ cd PATH_TO_YOURS_PROJECTS
iMac$ mkdir imega
iMac$ cd imega
iMac$ git clone git@github.com:iMega/boot2docker.git
```

## Start the machine

```
iMac$ vagrant up
```

## Stop the machine

```
iMac$ vagrant halt
```

## Remove the machine

```
iMac$ vagrant destroy
```
